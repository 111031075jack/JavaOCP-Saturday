4. 主程式解析邏輯
輸入格式：
- "1"           → 1號飲料，1杯
- "1x2"         → 1號飲料，2杯  
- "3x3,7x1,5"   → 3號3杯、7號1杯、5號1杯

解析步驟：
1. split(",") → ["3x3", "7x1", "5"]
2. 每個項目：
   - 有 "x" → split("x") → [3, 3] → 3號，3杯
   - 沒 "x" → 數字，預設1杯

測試資料
String test1 = "1, 3x2, 7";           // 1杯珍珠 + 2杯冬瓜 + 1杯粉紅
String test2 = "9x2, abc, 2x3, 5";    // 錯誤數字 + 多份
String test3 = "1x3, 1x2";            // 同款多次合併？

預期輸出
顧客輸入：1, 3x2, 7
解析結果：
✓ 1 → 珍珠奶茶 $60 × 1杯 = $60
✓ 3x2 → 冬瓜茶 $50 × 2杯 = $100
✓ 7 → 粉紅淑女 $65 × 1杯 = $65
✗ 9 → 超出範圍
